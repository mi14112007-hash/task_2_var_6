Этапы реализации

Этап 1: Минимальный прототип с конфигурацией

Загрузка конфигурации из INI-файла
Валидация параметров
Обработка ошибок
Этап 2: Сбор данных

Поддержка формата пакетов Rust (Cargo)
Получение зависимостей по версии пакета
Режим тестирования с файлом
Этап 3: Основные операции

Алгоритм BFS с рекурсией для построения графа
Учет максимальной глубины анализа
Фильтрация пакетов по подстроке
Обработка циклических зависимостей
Этап 4: Дополнительные операции

Определение порядка загрузки зависимостей
Сравнение с реальным менеджером пакетов
Этап 5: Визуализация

Генерация описания на языке Graphviz
Сохранение изображения графа
